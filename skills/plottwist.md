# /plottwist

A plot twist reframes everything the audience thought they understood.

## Rules
1. The twist must be **true** — not a trick, but a perspective shift
2. It should make the audience re-evaluate previous slides
3. Place it at ~40% of the deck (not the end — leave room to rebuild)
4. Signal it visually: change background, use a meme, pause

## Pattern
```
[Audience believes X]
  ↓
[Twist: X was actually Y all along]
  ↓
[Now re-read everything before with Y in mind]
```

## Example for this presentation
- Audience thinks: "This is a presentation about AI"
- Twist: "This presentation IS an AI agent execution. You're inside the context window right now."
- Rebuild: Now every slide is both content AND example
