# /creative-funny

Generate content that is:
- **Unexpected**: Break the pattern the audience anticipates
- **Relatable**: Reference shared developer experiences
- **Memorable**: Create a hook that persists after the presentation

## Strategy
1. Identify the core serious message
2. Find the absurd angle on it
3. Wrap the serious message inside the absurd framing
4. Ensure the humor reinforces (not undermines) the message

## Examples
- Topic "AI will replace developers" → "AI won't replace you. A developer who uses AI will."
- Topic "Memory management" → Start with a fish that can't remember (Dory)
- Topic "Context windows" → "Your RAM is 1M tokens. Your attention span is 3."
