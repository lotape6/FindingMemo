The basics

The Cast

The Model

A function: text in, text out. Stateless. No memory between calls. No ability to act. Pure reasoning -- incredibly smart, but alone.

Marlin's brain -- smart but alone

The Agent

A model wrapped in a loop with tools. The model decides. The loop keeps going. The tools let it act.

Marlin with fins and eyes

The Context

Everything the model can see. Its working memory. When full, old info gets compacted. Not the model -- the context makes the difference.

Practical rules

The Mindset

01

Never prompt in chat

Write in an editor. Read it twice. Dehumanize -- configure a system.

02

Scripting has evolved

Create skills, agents, reusable context. Write instructions once, reuse forever.

03

Keep context congruent

Don't teach the model. Give it the right information. Your job is congruence.

04

Think first, then prompt

Every wasted token is a wasted thought. Plan before you execute.
