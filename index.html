The basics

The Cast

The Model

A function: text in, text out. Stateless. No memory between calls. No ability to act. Pure reasoning -- incredibly smart, but alone.

Marlin's brain -- smart but alone

The Agent

A model wrapped in a loop with tools. The model decides what to do. The loop keeps it going. The tools let it act on the world.

Marlin with fins and eyes

The Context

Everything the model can see. Its working memory. When full, old info gets compacted. The difference between powerful and weak? Not the model -- the context.

Marlin vs Dory -- same brain, different capacity

Practical rules

The Mindset

01

Never prompt in chat

Write in an editor. Read it twice. Dehumanize -- you're configuring a system.

02

Scripting has evolved

Create skills, agents, and reusable context. If you write instructions twice, make it a file.

03

Keep context congruent

Don't tell the model how to be smart. Give it the right information. Your job is congruence.

04

Think first, then prompt

Every wasted token is a wasted thought. Plan before you execute.
