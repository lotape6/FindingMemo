About This Platform

Finding Memo is a self-contained learning platform where every lesson is simultaneously teaching material and a live example of the concepts it covers. The slides, the theme, and even this page were created by an AI agent — demonstrating the very workflow you are here to learn.

Built with MARP for interactive slide decks and Claude Code as the agentic development tool. No frameworks, no build steps for the landing page — just clean HTML and CSS.
